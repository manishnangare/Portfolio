My Projects

Quantum based Paddy disease classification

The project focuses on the early detection of sheath blight and blast disease in paddy crops using Quantum machine learning (QML) with model accuracy of 99.4%. By analyzing images of healthy and infected paddy leaves, the model can accurately predict the presence of disease. Early detection enables timely interventions, helping to reduce crop losses, improve rice yield, and contribute to food security. Given the rising global hunger and the dependence of more than 90% of the world’s population on agriculture, there is a pressing need for accurate, early disease detection. Leveraging machine learning and computer vision for automated plant disease identification provides farmers with real-time predictions and actionable insights. This proactive approach helps optimize resource use, minimize crop losses, and ensure sustainable agricultural practices

Technologies: Python, Cirq, Tensorflow, Tensorflow Quantum, keras, OpenCV, Google Colab, Pandas, NumPy, Matplotlib, Scikit-learn and Git

MediScan : Eye disease classification

The Eye Disease Classification project is designed to develop an automated system that uses deep learning models to classify fundus images (retinal images) into various categories, such as diabetic retinopathy, glaucoma, cataracts, and normal eye conditions. The project aims to provide healthcare professionals with an AI-powered tool to assist in diagnosing common eye diseases more efficiently and accurately. This tool will enable the detection of eye diseases at earlier stages, helping reduce the risk of irreversible vision loss.

Technologies: HTML, CSS, Python, Tensorflow, keras, Jupyter Notebook, OpenCV, Flask, Pandas, NumPy, Matplotlib, Scikit-learn and Git
